# LiveMate mini

## 概要

LiveMate mini は、「ひとりでライブ現場に行くのがちょっと怖い…」という人向けの、
ライブ同行者募集用のシンプルな掲示板アプリです。
ライブ名・日付・会場名・ひとことコメントを入力すると、その場で募集カードが一覧に追加されます。

## 公開URL(UIデモ)

https://11kazu13.github.io/live-mate-mini/

※このURLはフロント確認用です。
投稿の保存・読み込みはローカルJSON Serverを起動している場合のみ動作します。

## 想定ユースケース

1. フォームに「ライブ名」「日付」「会場名」「ひとこと」を入力する
2. 「投稿する」ボタンを押すと、下の一覧に募集カードが追加される
3. 新しい投稿ほど上に表示される
4. 削除ボタンで削除可能
