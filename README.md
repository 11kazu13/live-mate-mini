# LiveMate mini – ver1.0

## 概要

**LiveMate mini** は試作品として開発した、  
「ひとりでライブ現場に行くのがちょっと怖い…」という人向けの  
**ライブ同行者募集用・シンプル掲示板アプリ**です。

ライブ名・日付・会場名・ひとことコメントを入力すると、  
その場で募集カードが一覧に追加される、最小構成のWebアプリになっています。

本リポジトリでは **JSON Server を用いたローカル動作版（ver1.0）** を収録しています。  
実際に **Supabase などのDBを用いて公開運用している版** は、別リポジトリで管理しています。

---

## 想定ユースケース

1. フォームに以下の情報を入力  
   - ライブ名  
   - 日付  
   - 会場名  
   - ひとことコメント  

2. 「投稿する」ボタンを押す  
3. 募集カードが一覧に追加される  
4. 新しい投稿ほど上に表示される  
5. 各カードの「削除」ボタンから投稿を削除できる

## 補足

本プロジェクトは、  
過去にライブアイドルとして活動していた際に聞いた  
「一人でのライブ参戦が不安」「現場に行きたいけど勇気が出ない」  
という声をきっかけに作成しました。

学習目的と実体験ベースの課題解決を兼ねた、小さなプロダクトです。
